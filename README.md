A responsive restaurant website built with **HTML, CSS, and JavaScript**. This project is a foundational front-end web development exercise that includes a home page, table reservation system, user profile form, and smooth navigation.  

This version was created before learning React to strengthen HTML, CSS, and JavaScript skills.  

---

## Features

- **Home Page:** Displays information about the restaurant, menu, and specials.  
- **Table Reservation System:** Users can select dates, times, and reserve tables.  
- **User Profile Form:** Allows users to input personal details.  
- **Navigation:** Smooth navigation between different sections of the website.  
- **Responsive Design:** Works well on desktop and mobile devices.  

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox/Grid for layout, responsive design)  
- JavaScript (DOM manipulation, form validation)
